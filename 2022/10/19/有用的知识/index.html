随时可能会被问到的《基础题》

前端开发

1、前端开发主要有哪些层？语言和功能是什么？

主要有三层，分别是结构层、样式层和行为层

结构层：html；搭建页面骨架，放置页面部件。

样式层：css；美化页面，搭建布局结构。

行为层：JavaScript；给页面添加交互效果，实现动态交互等。

2、JavaScript的编写位置是哪里？

JavaScript除了在NodeJS中外，只能在html中编写

3、JavaScript有哪些输出语句

alert( ); 弹窗弹出

console.log(); 控制台输出

4、什么是变量？如何定义变量？变量都有哪些命名规则？

什么是变量？

在程序的执行过程中，其值可以在一定范围内发生改变的量，也是存储数据信息的容器。

如何定义变量？

在javascript中可以使用关键字：const，var，let来定义变量

var：定义的变量可以修改，如果不初始化会输出undefined，不会报错。

const：定义的变量不可以修改，而且必须初始化，const定义的是一个恒定的常量

let：是块级作用域，函数内部使用let定义后，对函数外部无影响。

变量有哪些命名规则？

1、必须以字母、下划线、或者美元符$开头；

2、除开头外后面的部分可以；

3、虽然变量名不限制长度，但能表达清楚命名的含义即可；

4、变量名不可以和java的关键字冲突；

5、中文拼音也可以命名，但工作中千万别这么干。

5、什么是变量声明的提升？

详细戳我

简洁版如下

// 你写的
console.log(a);
var a = 12;
console.log(a);
// 执行的
var a;
console.log(a);
a = 12;
console.log(a);
// 关键词：预解析，预编译，只提升定义，不提升值。

6、基本类型和引用类型的区别

基本类型

1、基本类型有：undefined / null / number / string / boolean / bigint / symbol
2、基本类型数据放在栈内存中
3、基本类型数据是不可变的

引用类型

1、引用类型有：Object / Array / Function 等，本质上都是属于 Object
2、引用类型以：地址: 数据 的映射关系来进行存储，其中地址放在栈内存，数据放在堆内存，若两个或 N 个变量指向同一个地址，则共用一份数据。
3、引用类型数据是可变的
