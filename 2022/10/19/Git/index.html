Git 日常学习笔记

git基础指令

1、提交代码

1.1、修改前先拉取最新代码（尽量减少冲突）
$ git pull
1.2、修改后准备提交查看目前缓冲区状态
$ git status

1.3、把内容添加到本地 git 缓冲区中
# 添加全部
$ git add . 
# 添加部分
$ git add 文件名
1.4、提交代码，将修改推送到远程git仓库中
# 引用的部分为提交说明，必须填写
$ git commit -m "提交代码"
1.5、将本地的分支版本上传到远程并合并
# git push 的命令格式一般是
git push <远程主机名> <本地分支名>：<远程分支名>
例如：git push origin master:master

当然，一般情况下，我们都不用写后面的，直接 git push 即可。

$ git push
